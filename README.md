# Denver Social Nights

## Project Setup

Welcome to the Denver Social Nights project! Below are the steps to set up the project locally.

### Prerequisites
- Ensure you have [Node.js](https://nodejs.org/) installed.
- Install [Git](https://git-scm.com/) for version control.

### Clone the Repository
```bash
git clone https://github.com/stharpe98-tech/denver-social-nights.git
cd denver-social-nights
```

### Install Dependencies
```bash
npm install
```

### Run the Application
```bash
npm start
```

### Folder Structure
- `src/` - Contains source code
- `public/` - Contains static files like images and stylesheets
